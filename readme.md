# TSP 双拼

适用于 [RIME 输入法](https://rime.im/) 的双拼方案

具体双拼方案清查看 `TSP.xlsx`

对于 Windows 的 RIME 输入法用户，复制 `TSP.dict.yaml` 和 `TSP.schema.yaml` 到 `用户\AppData\Roaming\Rime\` 下即可

其中，字典来自[极光双拼](https://github.com/hosxy/rime-aurora-pinyin)
